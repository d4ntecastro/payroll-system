# Payroll Management System

A Python-based system designed to automate salary calculations, tax deductions, and employee payroll reporting.

## 🚀 Features (In Progress)

- [ ] Employee Database Management (SQLite)
- [ ] Automatic Tax & Deduction Logic
- [ ] Monthly Payroll Report Generation
- [ ] Export Payslips to PDF

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Database:** SQLite

## ⚙️ Setup

1. Clone the repo: `git clone https://github.com/d4ntecastro/payroll-system.git`
2. Run the main script: `python src/main.py`
